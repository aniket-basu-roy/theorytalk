TheoryTalk

Let's Talk Theory

Download this project as a .zip file Download this project as a tar.gz file

This is a weekly series of informal talks given by people enthusiastic about Theoretical Computer Science. The goal is to have an interactive session discussing papers that would broaden our common knowledge. The speaker should consider the background of the audience and spend ample time in laying the basics of the paper before going on to the knitty-gritties. Also note this is not a platform to prove one's smartness but mutually learn something new.
